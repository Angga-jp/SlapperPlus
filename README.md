# SlapperPlus
Add-on plugin for Slapper, using forms to create & manage Slapper entities

*`/slapperplus`*


# Requirements
PocketMine-MP ALPHA9/10

[Slapper](https://github.com/jojoe77777/Slapper)     
[FormAPI](https://github.com/jojoe77777/FormAPI)
